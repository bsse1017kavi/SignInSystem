Please follow these instructions:
(1) Copy the 'mydb.sql' folder to xampp for Windows
(2) Import the contents of data using sql command
	mysql -u username -p dbname < mydb.sql
Reminder: dbname should be of a database that exists in the system or created by you
(3) Run Apache server
(4) Run MySQL server
(5) Copy contents of the project in xampp/htdocs
(6) From your browser traverse to that directory from localhost
(7) Start from wlcome.html
